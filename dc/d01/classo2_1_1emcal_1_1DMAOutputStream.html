Detailed Description

Output stream of a payload to DMA raw files.

Author
Markus Fasel marku.nosp@m.s.fa.nosp@m.sel@c.nosp@m.ern..nosp@m.ch, Oak Ridge National Laboratory
Since
Nov 6, 2019

Stream of output payload with variable size to DMA pages in a binary raw file. The output payload can be larger than the size of a DMA page (default: 8 kB) the output is split into multiple pages. Page counter, memory size, page size and stop bit are handled internally and are overwritten in the raw data header provided. All other header information must be handled externally.
