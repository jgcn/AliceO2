Detailed Description

This is the lowest class of the intermediate Digit Containers, in which all incoming electrons from the hits are sorted into after amplification The structure assures proper sorting of the Digits when later on written out for further processing. This class holds the individual GlobalPad containers and is contained within the Row Container.
