Detailed Description

template<typename T>
class o2::framework::BoostSerialized< T >

Enforce Boost serialization for a type

Usage: (with 'output' being the DataAllocator of the ProcessingContext) SomeType object; BoostSerialized<decltype(object)> ref(object); output.snapshot(Output{}, ref);

  • or - output.snapshot(Output{}, BoostSerialized<decltype(object)>(object));

The existence of the serialized overload for the wrapped type can not be checked at compile time, a runtime check must be performed in the substitution for the BoostSerialized type.
